Setup Blog

Geplaatst door Matthias Antierens op 8 oktober 2019

Deze blog is volledig gemaakt in html, css en javascript. Ik heb deze gemaakt aan de hand van een free online template, die volledig open source is zodat ik geen problemen zou krijgen met de licentie.

Om een nieuwe post te maken moet ik steeds een nieuwe html pagina maken en deze toevoegen aan de html stuctuur.

Ik heb de site op Git gezet, want dit maakt het makkelijker om van op verschillende toestellen te werken aan de blog.

Dit is een voorbeeld van hoe de blog technisch in elkaar zit.
